# 🎮 Steam Online Tracker

Привет! 👋

Небольшой, но полезный плагин для [Millennium](https://millennium.web.app/), который показывает **актуальный онлайн игроков** прямо в библиотеке Steam.

Больше не нужно открывать SteamDB или лезть в центры сообщества — просто кликаешь на игру и сразу видишь, сколько человек в ней прямо сейчас.

![Steam Online Tracker — бейдж с онлайном в углу баннера](https://img.shields.io/badge/Steam-Online%20Tracker-171a21?style=for-the-badge&logo=steam&logoColor=white)

---

## ✨ Что умеет

- **Выглядит как родной.** Аккуратная плашка в правом нижнем углу баннера игры. Полупрозрачный фон, размытие (glassmorphism), плавные анимации при наведении — как будто это сделали сами Valve.
- **Работает незаметно.** Никаких всплывающих окон, лагов или консолей. Данные берутся напрямую с серверов Steam через встроенный HTTP-клиент и кэшируются на 5 минут.
- **Безопасно.** За это **не дадут VAC-бан**. Плагин меняет только интерфейс библиотеки и никак не трогает античит или файлы игр.

---

## 📦 Установка

1. Убедись, что у тебя стоит [Millennium для Steam](https://millennium.web.app/).
2. Скачай последний релиз из раздела [Releases](https://github.com/yogurtrastishka2008-droid/SteamOnline-plugin/releases).
3. Распакуй архив в папку плагинов:
   ```
   C:\Program Files (x86)\Steam\millennium\plugins\
   ```
   > **Важно:** внутри `plugins` должна появиться папка плагина (например `steamonline`), а уже в ней — файлы `plugin.json`, `backend/`, `frontend/` и `.millennium/`.
4. Перезапусти Steam.
5. Готово! Заходи в библиотеку, выбирай любую игру — онлайн появится на баннере. 🎉

---

## 🔄 Обновление

Просто замени содержимое папки плагина на файлы из нового архива и перезапусти Steam.

## 🛠 Как это устроено

Steam жёстко ограничивает сетевые запросы из своего встроенного браузера (CORS/CSP), поэтому обычный `fetch` из фронтенда не работает.

Мы решили это элегантно:

| Слой | Что делает |
|------|-----------|
| **Frontend** (React/TSX) | Рисует бейдж на баннере, вызывает бэкенд через `callable` |
| **Backend** (Lua) | Получает запрос от фронтенда, валидирует `appId`, делает нативный HTTP-запрос к Steam API и возвращает результат |

Бэкенд использует **встроенный HTTP-клиент Millennium** (`require("http")`), который работает внутри процесса Steam — без вызова внешних утилит вроде `curl.exe` или `wscript.exe`.

## 🔒 Безопасность

- Все входящие данные проходят строгую валидацию (только цифры в `appId`).
- Никаких внешних процессов, скриптов или системных вызовов.
- Никаких сторонних серверов — данные идут напрямую с `api.steampowered.com`.
- Открытый исходный код — можешь проверить каждую строчку.

## ❓ FAQ

**Могут ли за это забанить?**
VAC-бан — нет. VAC работает только внутри защищённых игр. Этот плагин не трогает файлы игр и античит. Формально модификация клиента нарушает SSA (пользовательское соглашение Steam), но Valve не преследует за косметические моды.

**Почему онлайн не показывается?**
Убедись, что плагин включён в настройках Millennium (иконка Millennium → Plugins). Если не помогло — перезапусти Steam.

**Данные точные?**
Да, они берутся из официального Steam Web API (`GetNumberOfCurrentPlayers`). Обновляются каждые 5 минут.
